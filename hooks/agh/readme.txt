Important 
=========
Users of versions before 2019/11, please note that, for technical reasons, the filename has changed
from AppGiniHelper.mine.js 
to   AppGiniHelper.min.js
Please update your <script>-include tag in hooks/header-extras.php

Installation
============
1. Copy AppGiniHelper.min.js into the hooks-directory of your application
2. Edit hooks/header-extras.php
3. Insert <script src="hooks/AppGiniHelper.min.js"></script>
4. Start coding
